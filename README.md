Это репозиторий с решением задачи для стажировки в JetBrains
(добавление поддержки формата .editorconfig)

****

Задача: определить тип отступов во входном файле

****

Ключевой момент, на который я опирался, заключается в том, что в Java отступы добавляются в основном внутри фигурных скобок.
Отслеживая их расположение, можно судить о том, каков стандартный размер отступа в файле.
Ещё один случай появления отступов - блок case внутри switch.
Его обработать было немного труднее :)

Остаются необработанными некоторые необычные ситуации, но они не оказывают значительного влияния на результат работы программы.
К тому же, они требуют гораздо более глубокий анализ кода, чем производится сейчас.

Проект написан на `IntelliJ IDEA 2018.1 x64`, версия Java `1.8.0_172`
